## Installation :

1. Clone the repository:

```sh
git clone https://github.com/HazemSarhan/dragon-ecommerce-frontend.git
```

2. Navigate into the project directory:

```sh
cd /dragon-ecommerce-frontend
```

3. Install dependencies:

```sh
npm install
```

6. Start the server:

```sh
npm run dev
```
